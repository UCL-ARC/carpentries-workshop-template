GitHub CLI

GitHub provides a command line interface (CLI) that we will use to simplify the authentication setup between our computer and GitHub's platform.

  • Windows
  • MacOS
  • Linux

Proceed to install Github CLI by downloading the installer.

The easiest way to install Github CLI is through homebrew. First you need to install homebrew:

  1. Open a terminal window

  2. Type 

    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

    Follow the installation prompts

  3. Once homebrew is installed, you can then install Github's CLI

    brew install gh

Proceed to install Github's CLI following the option according to your distribution.
